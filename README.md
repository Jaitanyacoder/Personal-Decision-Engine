Hello there,

This project is a simple decision-making web app built using Streamlit that helps users compare different options and choose
the best one based on their priorities. Users can add options by entering a name and category, then rate each option using 
sliders for **interest level**, **growth potential**, and **effort required**. The app calculates a score (interest + growth
− effort) for each option and ranks them automatically. You can view all saved options, delete specific ones, or clear everything.
Finally, by clicking **"Generate decision report"**, the app displays a ranked list and highlights the best option based on your 
inputs.

## 🧾 How to Use

1. Enter an option name and category  
2. Adjust the sliders based on your preference  
3. Click **"Add Option"** to save it  
4. Repeat for multiple options  
5. Manage options (delete or clear if needed)  
6. Click **"Generate decision report"** to see the best choice
